#### 1) 상위 폴더명 : BAEKJOON(백준 온라인 저지)

|   폴더명   |                      목적                       |     
| :------: | :-----------------------------------------------:|
| 1000~ |                1000번대 이상 문제 풀이                |
| 10000~ |              10000번대 이상 문제 풀이                |
| 단계별 문제 |   백준 단계별 문제 풀이(단계를 하위 폴더로 구분)   | 

* 단계별 문제에서 아직 안 올라온 문제가 있다면 1000~ 또는 10000~ 폴더를 보시고, 만약 이 폴더 내에도 없다면 아직 풀지 않은 문제입니다.
